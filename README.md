# Pile
Pile is a mini-programming-language built in JavaScript, and mainly does math operations.

## Setup

- Install `Node.js`
- Clone the repo
##### HTTP
```sh-session
git clone https://github.com/loldonut/pile
```
##### SSH
```sh-session
git clone git@github.com:loldonut/pile
```

#### Then run it:
```bash
cd pile
# Links it to PATH
npm link
pile
```
